Sistema de Gerenciamento de Pacientes

Projeto desenvolvido para disciplina de Banco de Dados.

Tecnologias utilizadas:
- PHP
- PostgreSQL
- HTML

Funcionalidades:
- Cadastro de pacientes
- Consulta de pacientes
- Atualização de dados
- Exclusão de pacientes
- Controle de empréstimo de equipamentos
- Registro de devolução

O sistema realiza integração com banco de dados e controle de disponibilidade de equipamentos.


Como executar o projeto:

1. Instalar XAMPP
2. Importar o banco de dados
3. Colocar a pasta no diretório htdocs
4. Acessar localhost
